后台管理小项目总结

总结

最近在遵循视频教程学习一个后台管理系统的前后端分离项目,所以它最大的优点就是前后端分离了，在这个项目实战中我获得了很多，也知道了自己的短板和学习中的问题，但是，目前我还是没有任何可行的办法去克服我的这些短板，所以为了尽早可以让我有一个质的突破，我下定决定去写一写这个项目的总结，开始正题：

项目名称

xxx后台管理

目录结构

    • components (folder)
      • Dialog.vue

技术栈
  • 前端
    • Vue.js(vue-cli 3)
    • Vuex
    • Vue-router
    • element-ui
    • axios
    • jwt-decode token解析
  • 后端
    • Node.js
    • express
    • mongoDB
    • mongoose
    • body-parse 格式化request过来的数据，方便使用
    • jsonwebtoken 生成token
    • passport token验证
    • passport-jwt
      功能
  • 登陆，注册
  • 资金管理(CURD)
    虽然功能很少，但是在这个项目中我还是学到了很对东西，对vue和node算是有了更深一层的理解
    遇到的问题
  • 前端
    • UI组件的使用。究其根本应该对element-ui文档不熟悉
    • 布局。鬼知道发生了什么，居然在布局方面我花了大量的时间去调整一个登陆和注册页面的问题。
    • Vuex的使用和概念
    • Vue-router 路由守卫
    • axios 请求拦截 响应拦截(重点)
  • 后端
    • 跨域
    • express 路由的使用
    • token的生成和使用
    • token的验证
    • mongoose schema
要完善的问题
